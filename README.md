# TimeSeriesCourse
 
## НАЗНАЧЕНИЕ

Данный репозиторий размещен по адресу https://github.com/mzym/TimeSeriesCourse/ и содержит материалы учебного курса "Анализ и прогнозирование временных рядов методами искусственного интеллекта".
